# simplePyTable
A small code with which you can create tables with ASCII characters.
There are three versions. A German one with print(), an English one with print() and an English one without print().

Example: 
en_version_Tabelle.py

Number of columns: 3
Column width: 15
Number of rows: 3
Row height: 1
print("╔═══════════════╦═══════════════╦═══════════════╗")
print("║               ║               ║               ║")
print("╠═══════════════╬═══════════════╬═══════════════╣")
print("║               ║               ║               ║")
print("╠═══════════════╬═══════════════╬═══════════════╣")
print("║               ║               ║               ║")
print("╚═══════════════╩═══════════════╩═══════════════╝")
